# Git+CLI Workshop / Git+CLI 워크숍

A 3-hour hands-on workshop for absolute beginners learning git and the command line through AI-native tools.

AI 도구를 활용하여 git과 커맨드라인을 배우는 3시간 실습 워크숍입니다.

## Workshop Setup / 워크숍 준비

This workshop uses two repos:

이 워크숍은 두 개의 저장소를 사용합니다:

| Repo | Purpose |
|------|---------|
| **This repo** (`git-cli-workshop`) | Teaching materials — read these docs to learn |
| [**Demo repo**](https://github.com/isnbh0/git-cli-workshop-demo) (`git-cli-workshop-demo`) | Practice playground — do your git exercises here |
